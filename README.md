# Portfolio

경영학부 재학 중 컴퓨터공학 전과를 목표로 학습한 **전공 강의 복습 노트** 및 **프로젝트 소스코드** 아카이브입니다.
단순히 코드를 작성하는 것에 그치지 않고, 매주 배운 이론 내용을 정리하며 CS 기본기를 다졌습니다.

## 1. 강의 복습 및 이론 정리
수업 시간에 학습한 C언어, Java 프로그래밍의 핵심 이론을 정리한 노트입니다.
* **C Language:** 포인터, 메모리 관리, 구조체 등 기초 문법 정리
* **Java:** 객체 지향 프로그래밍(OOP) 개념, 컬렉션 프레임워크, 예외 처리 등 학습 기록
* **Path:** `C/`, `Java/` 폴더 내 정리 파일 참조

---

## 2. 실습 프로젝트
이론을 바탕으로 구현한 주요 프로젝트 소스코드입니다.

### [Project 1] Java Parking Management System
* **Path:** `Projects/1_ParkingSystem`
* **Description:** 객체 지향 설계를 적용한 주차 관리 시뮬레이터
* **Key Tech:**
  - Car - ParkingLot 클래스 분리를 통한 객체 지향 설계
  - 2차원 배열 활용 및 예외 처리(IndexOutOfBounds 방지)

### [Project 2] Vocabulary Master (Ver.1 Console)
* **Path:** `Projects/2_VocaMaster_Console`
* **Description:** Java 컬렉션 프레임워크를 활용한 영단어 암기 프로그램
* **Key Tech:**
  - HashSet을 활용한 중복 없는 퀴즈 알고리즘 구현
  - 기존 오답 노트와 신규 오답 병합 로직 구현

### [Project 3] Vocabulary Master (Ver.2 GUI)
* **Path:** `Projects/3_VocaMaster_GUI`
* **Description:** Ver.1의 로직을 Swing 프레임워크로 이식하고 팀원 코드를 통합(Integration)한 프로젝트
* **Key Tech:**
  - **Architecture:** `VocManager`와 `JPanel` 분리를 통한 유지보수성 확보
  - **Layout:** `JTabbedPane` 및 `CardLayout`을 활용한 직관적인 화면 구성
  - **Collaboration:** 팀원별 분산된 코드를 취합하고 충돌을 해결하는 메인 개발자 역할 수행
